# 织法·向量记忆 (Weaver Vector Memory)

这是为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 开发的第三方扩展插件，专门适配「织法者4.0【liteC4】」预设。
它能够自动读取 AI 回复中的 `<VEC_ARCHIVE>` 归档指令，自动存取和检索长期记忆，让 AI 能够"回想起"之前的细节。

## 📦 安装方法

### 方式一：GitHub 链接安装（推荐）
1. 打开 SillyTavern，进入 **扩展** 面板（顶部的魔方图标）
2. 展开 **安装扩展** 菜单
3. 在文本框中粘贴本插件的 GitHub 仓库地址（待定）
4. 点击安装并重启 SillyTavern

### 方式二：手动 ZIP 安装（无法访问 GitHub 的用户）
1. 下载本仓库的 `weaver-vec-memory.zip` 压缩包
2. 解压得到 `weaver-vec-memory` 文件夹
3. 将整个文件夹复制到你的 SillyTavern 安装目录下的：
   `public/scripts/extensions/third-party/`
4. 重启 SillyTavern
5. 在扩展面板中确认已勾选启用

## ⚙️ 核心功能与配置

安装完成后，在扩展面板最底部找到 **织法·向量记忆 (Weaver Memory)** 设置抽屉。

### 1. 两种检索模式
- **关键词匹配（默认推荐）**：无需下载任何模型，无需配置 API，即装即用。完全本地执行，速度极快。
- **向量语义检索**：支持对接硅基流动（SiliconFlow）等兼容 OpenAI 格式的外部向量 API，实现更高级的语义相似度检索。

### 2. 自动化机制
扩展会自动监听 AI 的回复。当 AI 按照预设在回复末尾输出 `[VEC]` 标签时，扩展会自动截获并保存到本地记忆库。
**你不需要进行任何手动操作。**

### 3. 可调参数
- **单轮最大检索量**：每次对话最多召回几条历史记忆（默认 5 条）
- **记忆衰减率**：每轮对话后，未被提及的记忆权重下降比例（默认 2%）。当记忆被检索命中时，权重会恢复。
- **归档重要度阈值**：过滤掉 AI 记录的低价值日常废话（默认 3 分，满分 10 分）。
